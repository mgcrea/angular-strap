mgcrea.ngStrap

About the project

AngularStrap is a set of native directives that enables seamless integration of Bootstrap 3.0+ into your AngularJS 1.2+ app.

With no external dependency except the Bootstrap CSS styles, AngularStrap is lighter and faster than ever as it does leverage the power of ngAnimate from AngularJS 1.2+!

The last stable release (v2.0) has landed in April 2014 after nearly 4 months of testing, during this beta, 365 commits landed from 17 contributors! Thanks to all the testers and contributors that made this release happen.

Quick Start

Install and manage AngularStrap with Bower. You can also use NuGet.

      
        $ bower install angular-strap --save

Load the required javascript libraries (Bootstrap scripts files are not needed).

      
        <script src="//ajax.googleapis.com/ajax/libs/angularjs/1.2.9/angular.min.js"></script> <script src="bower_components/angular-strap/dist/angular-strap.min.js"></script> <script src="bower_components/angular-strap/dist/angular-strap.tpl.min.js"></script>

Inject the mgcrea.ngStrapmodule into your application.

      
        angular.module('myApp', ['mgcrea.ngStrap']);

Custom builds

AngularStrap provides independently built modules that can be loaded separately

        
          angular.module('myApp', [ 'mgcrea.ngStrap.modal', 'mgcrea.ngStrap.aside', 'mgcrea.ngStrap.tooltip' ]);

Contribute

Build and work on AngularStrap with Gulp.

      
        // Serve and watch docs, ideal to hack!
        $ gulp serve
        // Continuous integration
        $ gulp karma:server
        // Build AngularStrap
        $ gulp build

Animations

AngularStrap leverages AngularJS ngAnimatemodule to provide animations to the directives. Therefore, it requires to load custom CSS code as ngAnimatecomes with a very specific markup.
