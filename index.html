Modal directives/modal.js

Overview and examples

Fetches an external html partial and populates the modal with its content.

Use any button to trigger a modal by appending a bs-modal tag.

Both $scope.show() and $scope.dismiss() are available inside the modal to toggle its visibility.

modal: {{modal | json}}
<!-- Button to trigger modal -->
<button type="button" class="btn" bs-modal="'partials/modal.html'">...</button>

<!-- Modal (./partials/modal.html) -->
<div class="modal-header">
  <button type="button" class="close" data-dismiss="modal" aria-hidden="true">×</button>
  <h3>Modal header</h3>
</div>
<div class="modal-body">
  <p>{{modal.content}}</p>
  <br />
  <pre>2 + 3 = {{ 2 + 3 }}</pre>
</div>
<div class="modal-footer">
  <button type="button" class="btn" ng-click="dismiss()">Close</button>
  <button class="btn btn-primary" ng-click="modal.saved=true;dismiss()">Save changes</button>
</div>
Heads up! You can use ng-controller="MyModalCtrl" along bs-modal="..." to create a custom scope for your modal.

Popover directives/popover.js

Overview and examples

Fetches an external html partial and populates the popover with its content.

Use any button to trigger a popover menu by appending a bs-popover tag.

Both $scope.show() and $scope.dismiss() are available inside the popover to toggle its visibility.

popover: {{popover | json}}
<!-- Button to trigger popover -->
<button type="button" class="btn" bs-popover="'partials/popover.html'">...</button>

<!-- Force close others with data-unique attribute -->
<button type="button" class="btn" data-unique="1" bs-popover="'partials/popover.html'">...</button>

<!-- Popover (./partials/popover.html) -->
<form name="popoverForm">
  <p>{{popover.content}}</p>
  <br />
  <pre>2 + 3 = {{ 2 + 3 }}</pre>
  <div class="form-actions">
    <button type="button" class="btn" ng-click="dismiss()">Close</button>
    <button class="btn btn-primary" ng-click="popover.saved=true;dismiss()">Save changes</button>
  </div>
</form>
Heads up! You can use ng-controller="MyPopoverCtrl" along bs-popover="..." to create a custom scope for your popover.

Button directives/button.js

Overview and examples

Enables databinding between a button and an scope object.

Use any button with ng-model and bs-button tags.

button: {{button | json}}
<button type="button" class="btn" ng-model="button.active" bs-button>...</button>

Checkboxes

checkbox: {{checkbox | json}}
<div class="btn-group" data-toggle="buttons-checkbox">
  <button type="button" class="btn" ng-model="checkbox.left" bs-button>...</button>
  <button type="button" class="btn" ng-model="checkbox.middle" bs-button>...</button>
  <button type="button" class="btn" ng-model="checkbox.right" bs-button>...</button>
</div>

Datepicker directives/datepicker.js

Overview and examples

Directive for bootstrap-datepicker (made by Stefan Petre).

Use any button to trigger a datepicker menu by appending a bs-datepicker tag.

✓ Supports AngularJS validation out of the box.

datepicker: {{datepicker | json}}
<div class="control-group input-append">
  <input type="text" ng-model="datepicker.date" data-date-format="dd/mm/yyyy" bs-datepicker>
  <button type="button" class="btn" data-toggle="datepicker"><i class="icon-calendar"></i></button>
</div>
Heads up! You can use a custom button as toggler with data-toggle="datepicker" on a sibling element.

Overview and examples

timepicker: {{timepicker | json}}
<div class="control-group">

  <i class="icon-time" style="position: relative;margin: 2.5px 0 0 -24px;"></i>
</div>
